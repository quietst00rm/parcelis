# PARCELIS

PARCELIS is a shipping insurance platform that protects customers and streamlines operations with licensed coverage backed by The Hartford.

## Project Setup

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

This project is built with:

- **Vite** - Build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework

## Development

The development server runs on `http://localhost:8080` with hot module replacement enabled.

## Building

To build the project for production:

```sh
npm run build
```

The built files will be in the `dist` directory.
